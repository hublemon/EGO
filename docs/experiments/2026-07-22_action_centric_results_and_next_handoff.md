## 0. 세 줄 요약

1. **candidate CE(Exp-A)가 acc 0.249 → 0.386으로 프로젝트 최고를 만들었다.** 평가 지표를 직접 겨냥한 첫 완주 학습이고, G1 보존·G2 교정을 동시에 올린 유일한 학습이다. 그러나 **무학습 기준선 L0(0.3994)에 0.013 미달**한다.
2. **L0 돌파를 위한 두 노선이 모두 실측으로 기각됐다.** G1 앵커는 "선택적 신뢰"가 전역 prior 이동으로 번져 순손실(−0.006), Retrospection 증류는 학습된 모델의 미래 headroom이 heldout에서 0(−0.003)이라 증류할 것이 없다.
3. **남은 최선 수는 추론 시 WM prior 융합이다** — 모델(G2 강함)과 L0(G1 완벽)의 상보성을 α-블렌드로 합친다. 학습 0, dev split 필요, 기대 acc ≈ 0.44.

## 2. L0 — 정의, 중요성, 왜 못 넘었나

**정의**: 학습 없이 WM top-1 후보를 그대로 따르는 정책. heldout acc **0.3994** (= P(G1)).

**중요성**: 이 선을 못 넘으면 학습의 순효과가 음수다. 이 기준선이 결과표에 없어서 "3일간 모든 학습이 무학습보다 못했다"는 사실이 늦게 발견됐다. **이후 모든 표에 L0 행을 강제한다** (`decompose_g1g2.py`가 자동 출력).

**왜 못 넘었나**: L0는 정의상 G1에서 1.000이다. Exp-A는 G1에서 0.671 — WM이 맞힌 것의 33%를 망친다. 산수로: `0.399×0.671 + 0.227×0.520 = 0.386`. L0를 넘으려면 조건부 정확도 > 0.638이 필요한데 0.617에 그쳤다.

## 3. 기각된 노선들 (전부 실측 근거)

### 3.1 belief 계열 (07-21, P3로 종결)
- margin 없는 consistency loss: cons_loss +10.7 → −13.85 폭주, reward 붕괴
- hinge(0.5) 추가: 폭주는 막았으나 (cons 0.26~0.37에 안착) **복창률 0.072 → 0.346 (4.8배)** — 모델이 belief에 답을 써넣는 shortcut으로 목적함수를 만족
- 복창 제외 ③ 재집계: 기존 런들의 복창 가설은 **기각** (wm ③ 0.0255→0.0260) — 그러나 복창을 보상하는 목적함수를 넣자 복창이 5배가 됨. 즉 복창은 원래 있던 게 아니라 그 목적함수가 만든 것
- 상세: `2026-07-21_p3_belief_redesign_future_plan_handoff.md`

### 3.2 G1 앵커 (Exp-C, 07-22)
`L = CE + kw·[G1]·KL(p_WM‖p_θ)`, kw ∈ {0.3, 1.0}:

| | acc | G1 | G2 |
|---|---:|---:|---:|
| Exp-A (kw=0) | 0.3860 | 0.6714 | 0.5202 |
| kw=0.3 | 0.3804 | 0.6678 | 0.5016 |
| kw=1.0 | 0.3797 | 0.6767 | 0.4829 |

**기각 근거**: 용량-반응이 명확하다 — kw↑에 G1 +0.009 / G2 −0.019. **G1/G2 구분은 입력에서 추론 불가능**하므로(WM top-1의 정오는 정답을 알아야 판별) 샘플 조건부 앵커가 전역 "WM을 믿어라"로 번진다. `0.399×ΔG1 + 0.227×ΔG2 < 0`. top-1 margin 등 변형도 같은 이유로 샐 것으로 판단, 방향 폐기.

### 3.3 Retrospection 증류 (3단계 관문, 07-22)
teacher headroom 3-arm 측정 (present / **실제 미래** / 남의 미래, 누설 차단: 미래에서 GT 동일 행동 전부 제거):

| teacher | 표본 | present | future | shuffled | 미래−현재 |
|---|---|---:|---:|---:|---:|
| base | train 1,370 | 0.3876 | 0.4314 | 0.3876 | **+0.0438** CI[+.023,+.065] |
| Exp-A | ~~train 1,370~~ | 0.5511 | 0.5344 | 0.5393 | (무효 — 학습 샘플 위 측정) |
| **Exp-A** | **heldout 307** | 0.5896 | 0.5863 | 0.5765 | **−0.0033** CI[−.049,+.042] |

**기각 근거**: base에서의 +0.044(대조군 정확히 0.000 — 깨끗한 통제)는 유효하나, **candidate CE 학습 후에는 heldout에서도 잔여 headroom이 0**이다. 해석: **GT 라벨이 곧 미래의 최대 압축**이므로(실현된 궤적의 다음 행동 = 라벨) GT 감독이 미래 suffix의 정보를 흡수·초과한다. 증류할 것이 없다.

### 3.4 논문 서사에 남는 것
1. 미래 궤적은 결정 관련 정보를 담는다 (+0.044, 통제됨) — frozen 모델 분석 결과로 유효
2. 그 정보의 가장 효과적인 투영 경로는 **action-level 감독**이다 — belief 텍스트·미래 조건부 target을 모두 이김
3. Retrospection 정의를 "완료된 궤적의 정보를 action credit으로 이전 결정 시점에 투영"으로 확정 (원안 §8과 합치)

## 4. 다음 수 — WM prior 융합 (권장)

**근거 (상보성)**: 모델 G2 0.520 / L0 G2 0.000 · L0 G1 1.000 / 모델 G1 0.671.

**방법**: 추론 시 `s_final(a) = s_θ(a) + α·log p_WM(a)`. α는 **train에서 뗀 dev split로 선정** (heldout 선정 금지 — test 피팅). G1 0.85 / G2 0.45만 실현돼도 acc ≈ 0.44.

**비용**: 학습 0. dev 분리 + 후보 스코어 전체 덤프(`eval_candidate_scored.py`에 5점수 저장 플래그 추가) + α 스윕은 재추론 없이 산술 = **약 1h**.

**주의**: 이는 Exp-C가 실패한 selective trust를 추론에서 구현하는 것이다 — G1/G2 판별이 필요 없고, WM 확률이라는 연속 신호를 직접 쓴다. 원안 §8(residual)·§9(gate)의 취지와 같다.

**보조 수**: 2-epoch Exp-A (63분, train_acc 0.60 vs train 상한 0.766로 여지 있음. 융합과 독립적이라 병행 가능).

## 7. 이번 라운드의 방법론적 실수 (반복 금지)

1. **L0를 결과표에 넣지 않았다** → 학습 순효과 음수가 3일간 은폐. headroom 실험에서도 재발 → 이후 모든 도구에 강제 출력.
2. **부분 데이터로 방향을 말했다가 두 번 뒤집혔다** (cw 스윕 3점/6점) → 판정 최소 표본을 코드에 박음.
3. **첫-끝 통계로 게이트를 만들었다** — 스스로 비판한 추정량을 다음 체인에 그대로 넣어 정상 학습을 죽임(2단계 1차). 참조 run의 급락 지점(37 step) 이전에 판정한 것도 오류.
4. **학습 샘플 위에서 headroom을 측정** → 무효 데이터 25분. heldout으로 재측정해 정정.
5. **train reward 하락을 heldout 열화로 단정** (P3) → 실제 heldout acc는 불변, 진짜 변화는 복창 4.8배.
6. **같은 그래프에 backward 2회** (Exp-C 1차) → 30샘플 crash-check를 본실행 전 표준 절차로.
7. 체인에 알림 미부착 → GPU 5.5h 유휴. 모든 체인에 Monitor 부착.
